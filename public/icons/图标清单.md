# macOS 图标清单

## 📍 图标位置
所有图标应放置在：`static/icons/` 目录下

## 📋 需要的图标文件

### 系统图标
- `apple-logo.png` - Apple Logo (14x14px，用于菜单栏)
- `finder.png` - Finder应用图标 (48x48px 或 64x64px)
- `trash.png` - 废纸篓图标 (48x48px 或 64x64px)

### 应用图标
- `safari.png` - Safari浏览器 (48x48px)
- `calculator.png` - 计算器 (48x48px)
- `calendar.png` - 日历 (48x48px)  
- `notes-improved.svg` - 备忘录 (48x48px, 矢量图标)
- `notes.png` - 备忘录 (48x48px, 原版图标)
- `settings.png` - 系统偏好设置 (48x48px)

## 🎨 图标规格要求

### 格式
- **推荐格式**: PNG (支持透明背景)
- **备选格式**: SVG (矢量图，可自动缩放)

### 尺寸
- **Dock图标**: 48x48px 或 64x64px
- **菜单栏图标**: 14x14px 或 16x16px  
- **桌面图标**: 48x48px 或 64x64px

### 风格要求
- **圆角**: 12px圆角（macOS标准）
- **阴影**: 自然投影效果
- **配色**: 符合macOS设计规范
- **透明度**: 支持透明背景

## 🔍 获取图标的方式

### 方式1: 从macOS系统提取
1. 在Mac上找到应用程序
2. 右键 → 显示包内容 → Contents → Resources
3. 找到 .icns 文件并转换为PNG

### 方式2: 在线图标库
- **Iconduck**: https://iconduck.com/
- **Icons8**: https://icons8.com/
- **Flaticon**: https://www.flaticon.com/
- 搜索关键词: "macOS", "Big Sur", "Monterey"

### 方式3: 设计工具
- **Figma**: 搜索macOS UI Kit
- **Sketch**: 下载macOS设计资源
- **Adobe XD**: 使用macOS组件库

## 🚀 快速替换步骤

1. **下载图标**: 按上述规格要求获取图标
2. **重命名**: 确保文件名与清单一致
3. **放置位置**: 复制到 `static/icons/` 目录
4. **刷新页面**: 重新运行项目查看效果

## 💡 临时解决方案

如果暂时没有合适的图标，可以：
1. 使用emoji作为临时图标
2. 使用纯色方块作为占位符
3. 先用文字标签代替

## 📂 目录结构示例
```
static/icons/
├── apple-logo.png      # Apple标志
├── finder.png          # 文件管理器
├── safari.png          # 浏览器
├── calculator.png      # 计算器
├── calendar.png        # 日历
├── notes.png           # 备忘录
├── settings.png        # 系统设置
└── trash.png           # 废纸篓
```

完成图标替换后，项目会自动使用新图标！
