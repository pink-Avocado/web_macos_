# 开发新应用指南

本文档详细说明如何在 macOS Web 项目中添加一个新的应用。

## 概述

添加新应用需要完成以下几个步骤：
1. 创建应用组件
2. 注册应用配置
3. 添加国际化翻译
4. 准备应用图标
5. 在主应用中注册路由

---

## 步骤详解

### 1. 创建应用组件

在 `src/components/apps/` 目录下创建新的 Vue 组件文件。

**文件位置：** `src/components/apps/YourApp.vue`

**基础模板：**

```vue
<template>
  <div class="your-app">
    <div class="app-header">
      <h1>应用标题</h1>
    </div>
    
    <div class="app-content">
      <!-- 你的应用内容 -->
    </div>
  </div>
</template>

<script setup lang="ts">
import { ref, onMounted } from 'vue'

// 接收传入的 props（如果需要）
const props = defineProps<{
  // 定义你需要的 props
}>()

// 你的应用逻辑
onMounted(() => {
  console.log('应用已加载')
})
</script>

<style scoped>
.your-app {
  width: 100%;
  height: 100%;
  display: flex;
  flex-direction: column;
  background: var(--window-bg);
  color: var(--text-primary);
}

.app-header {
  padding: 20px;
  border-bottom: 1px solid var(--border-color);
}

.app-content {
  flex: 1;
  overflow: auto;
  padding: 20px;
}

/* 支持深色模式的样式变量已在全局定义 */
</style>
```

**注意事项：**
- 使用 CSS 变量（如 `var(--window-bg)`）以支持深色模式
- 组件应该是响应式的，能够适应不同的窗口大小
- 如果需要访问其他应用或全局状态，使用 `inject` 获取

---

### 2. 注册应用配置

在 `src/config/apps.ts` 中添加应用配置。

**示例：**

```typescript
{
    id: 'yourapp',                    // 应用唯一标识符（小写，无空格）
    nameKey: 'apps.yourapp',          // 国际化键值
    icon: '/static/icons/yourapp.png', // 图标路径
    showInLaunchpad: true,            // 是否在启动台显示
    showInDock: true                  // 是否在程序坞显示
}
```

**配置说明：**
- `id`: 必须唯一，建议使用小写字母和下划线
- `nameKey`: 指向国际化翻译的键值路径
- `icon`: 图标文件路径，建议使用 PNG 格式，尺寸 512x512px
- `showInLaunchpad`: 设为 `true` 则应用会出现在启动台
- `showInDock`: 设为 `true` 则应用会固定在程序坞

---

### 3. 添加国际化翻译

在 `src/locales/zh-CN.ts` 中添加应用名称翻译。

**位置：** `apps` 对象内

```typescript
export const zhCN = {
    // ...其他配置
    apps: {
        finder: '访达',
        // ...其他应用
        yourapp: '你的应用名称',  // 添加这一行
        trash: '废纸篓'
    },
    // ...
}
```

**可选：** 如果应用有自己的翻译内容，可以添加专门的配置节：

```typescript
export const zhCN = {
    // ...
    yourapp: {
        title: '应用标题',
        button1: '按钮1',
        message: '欢迎使用'
    }
}
```

---

### 4. 准备应用图标

**要求：**
- 格式：PNG（推荐）或 SVG
- 尺寸：512x512px（推荐）
- 背景：透明或圆角矩形
- 风格：尽量符合 macOS 设计语言

**存放位置：** `public/static/icons/yourapp.png`

**设计建议：**
- 参考现有应用图标的风格
- 使用清晰、简洁的图标设计
- 确保在小尺寸下也能清晰识别

---

### 5. 在主应用中注册路由

在 `src/App.vue` 中导入并注册你的应用组件。

**步骤 1：导入组件**

```typescript
import YourApp from './components/apps/YourApp.vue'
```

**步骤 2：在 `appComponents` 对象中注册**

```typescript
const appComponents: Record<string, Component> = {
  finder: FinderApp,
  safari: SafariApp,
  // ...其他应用
  yourapp: YourApp,  // 添加这一行
}
```

**步骤 3：（可选）添加默认窗口配置**

如果你的应用需要特定的默认窗口尺寸，可以在 `openApp` 函数中添加：

```typescript
function openApp(appId: string, props?: any) {
  // ...
  let defaultWidth = 800
  let defaultHeight = 600
  
  if (appId === 'yourapp') {
    defaultWidth = 1000
    defaultHeight = 700
  }
  // ...
}
```

## 完整示例：添加一个"天气"应用

### 1. 创建组件 `src/components/apps/WeatherApp.vue`

```vue
<template>
  <div class="weather-app">
    <div class="weather-content">
      <h1>天气</h1>
      <div class="temperature">{{ temperature }}°C</div>
      <div class="description">{{ description }}</div>
    </div>
  </div>
</template>

<script setup lang="ts">
import { ref } from 'vue'

const temperature = ref(22)
const description = ref('晴天')
</script>

<style scoped>
.weather-app {
  width: 100%;
  height: 100%;
  background: linear-gradient(to bottom, #4facfe 0%, #00f2fe 100%);
  display: flex;
  align-items: center;
  justify-content: center;
}

.weather-content {
  text-align: center;
  color: white;
}

.temperature {
  font-size: 72px;
  font-weight: 200;
  margin: 20px 0;
}
</style>
```

### 2. 在 `src/config/apps.ts` 中添加

```typescript
{
    id: 'weather',
    nameKey: 'apps.weather',
    icon: '/static/icons/weather.png',
    showInLaunchpad: true,
    showInDock: false
}
```

### 3. 在 `src/locales/zh-CN.ts` 中添加

```typescript
apps: {
    // ...
    weather: '天气',
    // ...
}
```

### 4. 准备图标

将天气图标放置在 `public/static/icons/weather.png`

### 5. 在 `src/App.vue` 中注册

```typescript
import WeatherApp from './components/apps/WeatherApp.vue'

const appComponents: Record<string, Component> = {
  // ...
  weather: WeatherApp,
}
```

## 高级功能

### 应用间通信

使用 `inject` 获取全局方法：

```typescript
const openApp = inject('openApp') as (appId: string, props?: any) => void

// 打开另一个应用
function openFinder() {
  openApp('finder')
}
```

### 接收外部参数

通过 props 接收参数：

```typescript
const props = defineProps<{
  path?: string
  data?: any
}>()

// 其他应用可以这样打开：
// openApp('yourapp', { path: '/some/path', data: {...} })
```

### 持久化数据

使用 `localStorage` 保存应用状态：

```typescript
import { ref, watch } from 'vue'

const appState = ref({
  // 你的状态
})

// 从 localStorage 加载
const saved = localStorage.getItem('yourapp_state')
if (saved) {
  appState.value = JSON.parse(saved)
}

// 监听变化并保存
watch(appState, (newVal) => {
  localStorage.setItem('yourapp_state', JSON.stringify(newVal))
}, { deep: true })
```

1. **使用浏览器开发者工具**
   - 检查控制台错误
   - 使用 Vue DevTools 调试组件状态

2. **测试不同场景**

   - 测试窗口调整大小

3. **性能优化**
   - 避免不必要的重渲染
   - 大数据列表使用虚拟滚动
   - 图片使用懒加载

## 常见问题

### Q: 应用图标不显示？
A: 检查图标路径是否正确，确保文件存在于 `public/static/icons/` 目录下。

### Q: 应用没有出现在启动台？
A: 检查 `apps.ts` 中的 `showInLaunchpad` 是否设为 `true`，并清除浏览器 localStorage 缓存。

### Q: 深色模式下样式异常？
A: 确保使用 CSS 变量而不是硬编码颜色值。参考 `src/style.css` 中定义的变量。

### Q: 如何设置应用的默认窗口大小？
A: 在 `App.vue` 的 `openApp` 函数中为特定应用 ID 设置 `defaultWidth` 和 `defaultHeight`。

3. ✅ 添加翻译（`src/locales/zh-CN.ts`）
